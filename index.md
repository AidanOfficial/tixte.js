# tixte.js

A wrapper for the Tixte API

## Install the library
```SH
npm i tixte.js
```

## Documentation
Initializing the client:
```JS
const Tixte = require("tixte.js");

const client = new Tixte({
  api_key: "your-key-here"
});
```
